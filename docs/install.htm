VTelegram
Simple yet powerful PHP library for Telegram Bot API
Installation

First things first, you need to download VTelegram.zip package on Downloads page.

Then require source files in your PHP code:

require_once '/path/to/VTelegram/VTelegram.php';
require_once '/path/to/VTelegram/VTgInlineKeyboard.php'; // e.g. if needed

Or require everything in VTelegram library with autoloader (in autoload.php):

require_once '/path/to/VTelegram/autoload.php';

You can even place autoloader into any folder, but then remember to define path to VTelegram files in VTELEGRAM_REQUIRE_DIR constant, e. g.:

define('VTELEGRAM_REQUIRE_DIR', '/path/to/VTelegram');
require_once '/path/to/autoloader/autoload.php';

Now you are ready to use Telegram Bot API via VTelegram!
